# 🎓 Faculty Availability System

A web application that allows students and staff to view faculty 
availability schedules easily.

## 📌 Features
- View faculty availability by day/time
- Search faculty by name or department
- Simple and clean UI

## 🛠️ Tech Stack
- React / HTML / CSS / JavaScript
- Built with Claude AI (Anthropic)
